Vario Library

Getting Started

Welcome to the Vario color and font library.
Vario means variable in both Italian and Spanish, and that's what this library aims to be. Vario is essentially a boilerplate, some optional fonts, and a color palate. This allows Vario to be as lightweight as possible, around 3kB in fact.
To get started just add the following two files to your project and edit the second one, which is all set up with preset colours etc.

Add Fonts

We've picked a number of go-to fonts to get you started. Example html links are provided for pnthe regular style only.

Heading fonts

Oswald

Owswald is san-serif typeface based on classic gothic and grotesque styles. It has a condensed, thick and tall appearance, perfect for bold headings. Pair Oswald with Merriweather, Roboto or Open Sans.
<link href="https://fonts.googleapis.com/css?family=Oswald" rel="stylesheet">

Abril Fatface

Abril Fatface is a modern serif typeface and is an elegant headline choice. Only the regular style is available for free from google fonts. Pair Abril Fatface with Open Sans or Roboto.
<link href="https://fonts.googleapis.com/css?family=Abril+Fatface" rel="stylesheet">

Playfair Display

Playfair Display is a transitional serif typeface. It is best used for titles and headlines and has a classy, feminine feel. Pair Playfair Display with Rubik or Open Sans.
<link href="https://fonts.googleapis.com/css?family=Playfair+Display" rel="stylesheet">

Lobster 2

Lobster 2 is a fun script typeface ideal for heaedings. Pair Lobster 2 with Roboto or Rubik.
<link href="https://fonts.googleapis.com/css?family=Lobster+Two" rel="stylesheet">

Roboto Slab

Roboto slab is a slab serif typeface. As it doesn't have any italic styles, I'd refrain from using it for body copy. Pair Roboto Slab with Roboto.
<link href="https://fonts.googleapis.com/css?family=Roboto+Slab" rel="stylesheet">

Paragraph fonts

Roboto

Roboto is a grotesque sans-serif typeface, and is the standard typeface for Android. It is very legible on screens of all sizes. Pair Roboto with Roboto Slab, Lobster 2 or Abril Fatface.
<link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">

Open Sans

Open Sans is a humanist sans-serif typeface. It is an extremely versatile font and is found all over the web. It has an open and friendly vibe, and is great used for body copy. Pair Open Sans with Merriweather, Oswald or Abril Fatface.
<link href="https://fonts.googleapis.com/css?family=Open+Sans" rel="stylesheet">

Merriweather

Merriweather is a serif typeface designed to be highly visible on screens. Pair Merriweather with Oswald. Merriweather also works beautifully as a heading typeface, and can be paired with Open Sans or Rubik body copy.
<link href="https://fonts.googleapis.com/css?family=Merriweather" rel="stylesheet">

Rubik

Rubik is a sans-serif typeface characterised by its rounded corners. Pair Rubik with Merriweather, Playfair Display or Lobster 2. Rubik also works well as a heading typeface, and can be paired with Roboto or Open Sans body copy.
<link href="https://fonts.googleapis.com/css?family=Rubik" rel="stylesheet">
