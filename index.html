VariableCSS Fonts and Colors

Here are the twelve colors of the VariableCSS framework.
To add a color, simply add the class listed on the box of that color.
They can also be used as text colors by removing the '-bg' from the class.

Tip: When adding multiple classes it's recommended to add the color class last to avoid specificity conflicts.

.magenta
.red
.orange
.amber
.yellow
.medium-green
.blue-green
.light-blue
.medium-blue
.dark-blue
.dark-grey
.light-grey

Note:Fonts types as classes and variables are not yet supported. Stay tuned for a future release.

Tips for choosing fonts:

  1. Keep it simple. Sans-serif fonts are usually a safe choice. Be especially careful with decorateive and script fonts.
  2. Avoid very similar looking fonts on the same website.
  3. Aim to use only a couple fonts, and make sure they convey a similar mood.

VariableCSS Cards

Creating a card is easy.
Create a div with a class of either 'card' or 'rounded-card' depending on the desired sharpness of the corners. Add the class to state the size of the card. Add a color.

Size classes are listed below:
.full-screen-card (takes up 95% of the viewport);
.max-screen-card (takes up 100% of viewport);
.half-screen-card (takes up half of viewport width);
.quarter-screen-card (takes up a quarter of the viewort width);
.eigth-screen-card (takes up an eigth of the viewport width);
.third-screen-card (takes up a third of the viewport width);
.fifth-screen-card (takes up a fifth of the viewport width)

Examples:

Classes:
.rounded-card
.quarter-screen-card

Classes:
.square-card
.third-screen-card

Classes:
.card
.half-screen-card

CSS and SASS Variable Colors

Now you can use CSS variables to update colors
The names of the variables are the same as the name names of the classes for those colors, except it says with '--'.

Examples are shown below:

background-color: var(--amber);
color: var(--dark-grey);
background-color: var(--magenta);
color: var(--light-grey);
background-color: var(--medium-blue);
color: var(--light-grey);

Note: CSS Variables have good support in most browsers, including Chrome, Firefox and Safari. However they are not yet supported in IE or Edge. Therefore, if using them, it is recommended to ensure the website is usable without CSS variables.
SASS variables have no known issues.
